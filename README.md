# riscy

A RISC-V emulator written in C++.

## Running

To run execute:

```sh
cmake -B out
cmake --build out
./out/riscy_emu
```

## Development

This is in very early stages of development and might not be stable enough to run actual software.
My ultimate milestone with this project would be to run the linux kernel.

## Naming

This project's name might change in the future. Calling it "riscy_emu" is most likely not a good idea since the name might be quite popular.
